# Domain kernels

Use these kernels as completeness checks. Select only the relevant domain.

## Medicine and health

definition -> epidemiology/risk factors when supplied -> anatomy/physiology basis -> pathogenesis -> clinical manifestations -> key tests -> diagnosis -> differential diagnosis -> treatment principles -> procedures/surgery if relevant -> contraindications/cautions -> complications -> follow-up/prevention -> high-risk warning

Do not invent doses, guideline years, recommendation grades, or patient-specific decisions. Mark missing source information.

## Mathematics

definition -> assumptions/conditions -> theorem/formula -> variable meanings -> derivation intuition -> geometric or algebraic interpretation -> problem types -> edge cases -> common calculation errors -> one-line discrimination from similar formulas

## Natural science

phenomenon -> observation -> mechanism -> law/model -> conditions -> variables -> prediction -> experiment/evidence -> limitations -> applications

## Engineering and technology

problem -> requirements -> constraints -> design principle -> architecture/components -> workflow/algorithm -> validation -> performance tradeoffs -> failure modes -> safety/security risks -> operational cautions

## Computer science and ai

task -> input/output -> representation -> model/algorithm -> training or execution steps -> complexity/resource limits -> evaluation metrics -> failure cases -> deployment constraints -> examples

## Law and policy

rule purpose -> legal source when supplied -> protected interest -> elements/conditions -> exceptions -> burden/standard if supplied -> legal consequence -> risk point -> similar rule contrast -> learning-only caution

Do not invent jurisdictions, current statutes, or legal consequences beyond the supplied source.

## Business, finance, and management

business problem -> objective -> actors -> variables/drivers -> decision logic -> metric -> implementation path -> tradeoffs -> risk -> feedback loop -> common misuse

For investment/finance, avoid personalized advice and separate conceptual learning from current market facts.

## Humanities and history

background -> actors/terms -> causes -> process -> key events/texts -> consequences -> interpretations -> controversy -> influence -> comparison -> evaluation boundary

## Language learning

meaning -> structure/rule -> use context -> examples -> register/style -> exceptions -> contrast with similar expression -> common learner errors -> practice transformation

## Practical skills

goal -> standard of good performance -> prerequisites -> steps -> key actions -> feedback signals -> failure modes -> correction method -> practice progression -> transfer boundary

## Exam review

scope -> high-frequency concepts -> discriminators -> formulas/rules -> typical traps -> recall anchors -> rapid review chain -> answer strategy
